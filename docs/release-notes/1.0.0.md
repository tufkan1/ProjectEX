# ProjectEX 1.0.0

ProjectEX 1.0 is the first stable Fabric release for Minecraft Java Edition 26.2.
It freezes the public integration API, configuration schema 1, and persisted-data
format 1 after the complete audit suite and public alpha release rehearsal.

## Compatibility

- Minecraft 26.2
- Fabric Loader 0.19.3 or newer
- Fabric API 0.155.2+26.2
- Java 25
- Optional Jade 26.2 integration

EMI is not included because its upstream project does not publish a Fabric 26.2
artifact. ProjectEX has no hard dependency on Jade, EMI, or another inspection mod.

## Upgrade and rollback

Back up the complete world and `config/projectex/` before upgrading. Servers coming
from `0.1.0-alpha.1` use the tested format-0 baseline migration: run
`/projectex migration dry-run`, create a migration backup, then verify the staging
server before production. ProjectEX refuses data newer than the running version and
can create an offline recovery package for rollback. See the server guide and
configuration/migration guide for the full procedure.

## Verification

The release contains byte-reproducible runtime, source, and Javadoc jars, SHA-256
checksums, and a CycloneDX 1.6 JSON SBOM. GitHub-hosted provenance and SBOM
attestations can be checked with:

```text
gh attestation verify projectex-fabric-1.0.0.jar --repo tufkan1/ProjectEX
```

See `CHANGELOG.md` for the complete feature list. Report ordinary defects with the
repository issue templates and security vulnerabilities through `SECURITY.md`.
