# ProjectEX 1.1.2

ProjectEX 1.1.2 completes EMC coverage for the Minecraft 26.2 content catalog.

## Highlights

- Every registered ProjectEX item now receives an EMC value from the bundled
  ProjectE-compatible seeds and the server's recipe graph.
- Dynamic Klein, Magnum, Colossal, and Gargantuan Star upgrades are mapped with their
  actual four-star-plus-catalyst recipe cost.
- Large tooltip values are compact and readable: `74488428` now appears as `74.4M`.
- A permanent dedicated-server coverage test prevents future ProjectEX items from
  shipping without EMC.

The release remains save-compatible with the stable 1.x line for Minecraft 26.2.
