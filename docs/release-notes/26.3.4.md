# ProjectEX 26.3.4 snapshot line

This prerelease rebuild brings the ProjectEX 1.1.6 JEI recipe-transfer work to all four
supported Minecraft 26.3 snapshot profiles:

- `projectex-fabric-26.3-snapshot-1-1.2.0-snapshot.1.jar`
- `projectex-fabric-26.3-snapshot-2-1.2.0-snapshot.2.jar`
- `projectex-fabric-26.3-snapshot-3-1.2.0-snapshot.3.jar`
- `projectex-fabric-26.3-snapshot-4-1.2.0-snapshot.4.jar`

The jars contain ProjectEX's optional JEI Fabric plugin entrypoint and Arcane Crafting
recipe-transfer bridge. JEI has not published a Minecraft 26.3 Fabric runtime at the time
of this release, so the integration cannot be activated or tested with JEI on these
snapshots yet. ProjectEX itself remains fully functional without JEI.

Each snapshot profile is independently compiled, tested, audited, and packaged. The release
contains runtime, source, Javadoc, CycloneDX SBOM, checksum, and provenance artifacts. Use
only the jar whose complete snapshot name matches the game. Snapshot builds are experimental;
back up the world before installing or updating.
