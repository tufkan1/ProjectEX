# ProjectEX 26.3.3 snapshot line

This prerelease rebuild brings the ProjectEX 1.1.5 controls and client preferences to all
four supported Minecraft 26.3 snapshot profiles.

- `V` changes charge level, `G` changes mode, and `K` opens an inventory Arcane Tablet.
- Every binding is remappable in Minecraft's native ProjectEX controls category.
- The optional Mod Menu settings entrypoint controls EMC tooltips, compact EMC numbers,
  remembered favorites, and transmutation-search focus.
- Arcane Tablet shortcut requests are server-validated and rate-limited.
- Every snapshot profile passed its unit suite, all 55 server GameTests, and JAR assembly.

The release contains separate runtime, source, and Javadoc jars for Snapshot 1, Snapshot 2,
Snapshot 3, and Snapshot 4. Use only the jar whose Minecraft snapshot name exactly matches
the game. Snapshot builds are experimental; back up the world before installing or updating.
