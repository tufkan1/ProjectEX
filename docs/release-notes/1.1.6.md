# ProjectEX 1.1.6

ProjectEX 1.1.6 adds native Just Enough Items integration for Minecraft 26.2.

JEI 30.13.0.84 now discovers ProjectEX through its Fabric plugin entrypoint. Opening an
Arcane Tablet in 3x3 crafting mode and viewing a crafting recipe in JEI displays the recipe
transfer `+` button. Pressing it fills the nine crafting inputs from all 36 player inventory
slots using JEI's standard transfer rules.

JEI remains optional and is not embedded in the ProjectEX jar. ProjectEX continues to boot
and work without JEI. The release matrix separately verifies:

- the full client journey with no optional integration;
- JEI plugin discovery and runtime construction;
- live resolution of the Arcane Crafting transfer handler;
- Jade and Mod Menu compatibility;
- all unit, GameTest, release-audit, reproducibility, checksum, SBOM, and provenance gates.

This release is save-compatible with ProjectEX 1.x for Minecraft 26.2. Back up worlds before
changing any mod set.
