# ProjectEX 1.1.3

ProjectEX 1.1.3 restores the original ProjectE machine inventory behavior and fuel family.

## Highlights

- Red Matter Furnace uses the original 13-input, one-fuel, 13-output layout and no longer
  writes products into invisible slots when its visible output is full.
- The furnace flame now reports remaining fuel, and the red tier matches ProjectE's
  three-tick cook interval.
- Alchemical Chest and Energy Condenser menus open with either an empty or occupied hand.
- Collectors upgrade coal into Alchemical Coal and continue through the complete fuel chain.
- Alchemical Coal, Mobius Fuel, and Aeternalis Fuel blocks include ProjectE visuals,
  recipes, exact EMC values, furnace fuel behavior, loot, tags, and translations.
- Existing matter-furnace inventories migrate to the new layout without deleting stacks.

The same fixes are included in the rebuilt Minecraft 26.3 Snapshot 1-4 prerelease artifacts.
