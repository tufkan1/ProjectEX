# ProjectEX 1.1.0

ProjectEX 1.1.0 is the visual-source and EMC-tooltip release for Minecraft 26.2.

## Highlights

- Original MIT-licensed ProjectE and ProjectExpansion item, block, animation, and GUI
  visuals are bundled from immutable upstream commits with full provenance and licenses.
- Every item with a server-published base EMC value shows that value beneath its tooltip.
  The cache is replaced on connection and after datapack/recipe reload; the client never
  invents EMC values.
- Existing ProjectEX keyboard, narrator, paging, ownership, and server-authoritative
  controls remain active over the original source panels.
- Artifact names now include both targets: `projectex-fabric-26.2-1.1.0.jar`.
- README includes the AI-development and user-risk notice in English and Turkish.

## Compatibility

- Minecraft Java Edition 26.2
- Fabric Loader 0.19.3+
- Fabric API 0.155.2+26.2
- Java 25

Back up worlds before changing any mod set. ProjectEX 1.1.0 remains on configuration
schema 1 and persisted-data format 1, so no data migration from 1.0.0 is required.

## Verification

Download only from the repository's GitHub release. Verify the runtime jar with:

```text
gh attestation verify projectex-fabric-26.2-1.1.0.jar --repo tufkan1/ProjectEX
sha256sum -c SHA256SUMS
```

The release also contains source/Javadoc jars, a CycloneDX SBOM, checksums, and GitHub
artifact attestations.
