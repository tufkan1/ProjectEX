# ProjectEX 26.3 snapshot line

This prerelease contains four independent Fabric builds:

- `projectex-fabric-26.3-snapshot-1-1.2.0-snapshot.1.jar`
- `projectex-fabric-26.3-snapshot-2-1.2.0-snapshot.2.jar`
- `projectex-fabric-26.3-snapshot-3-1.2.0-snapshot.3.jar`
- `projectex-fabric-26.3-snapshot-4-1.2.0-snapshot.4.jar`

Use only the jar matching the exact Minecraft snapshot. Every target is compiled against
its corresponding Fabric API and passes the complete dedicated-server game-test suite.
Snapshot 4 also carries compatibility for the new cooking-fuel, input, recipe-tag, and
loot-modifier APIs.

The rebuilt artifacts include the 1.1.3 machine-parity fixes: original matter-furnace
input/output layouts and flame progress, held-item Chest/Condenser opening, coal collector
upgrades, and the three ProjectE fuel blocks with recipes, EMC, loot, and source visuals.

Vanilla EMC coverage now combines the Tekkit Classic catalog with modern ProjectE seeds.
Craftable values are calculated from the active recipe graph, while otherwise uncraftable
1.21+ and 26.3 content has an explicit value. Back up worlds before trying snapshot builds.
