# ProjectEX 1.1.1

ProjectEX 1.1.1 is a visual-layout correction release for Minecraft 26.2.

## Fixes

- Energy Condenser MK1/MK2/MK3 and both alchemical chests use the complete original
  chest model for inventory and placed rendering, including horizontal placement direction.
- ProjectE and ProjectExpansion container textures render at their native dimensions.
- Slots are selected from the source layouts for each condenser, chest, collector, relay,
  and matter-furnace tier.
- Public/private, redstone, paging, and ProjectEX-only actions are separated from source
  panels so they cannot cover original slots or artwork.
- The ProjectExpansion alchemical book remains textureless and responsive like its source.
- ProjectEX automation configuration no longer presents an unrelated source texture as a GUI.

All valued, component-free items continue to show their authoritative EMC value in the
tooltip supplied by the server.

## Compatibility

- Minecraft Java Edition 26.2
- Fabric Loader 0.19.3+
- Fabric API 0.155.2+26.2
- Java 25

Configuration schema 1 and persisted-data format 1 are unchanged. Back up worlds before
changing any mod set.

## Artifact

The primary runtime artifact is `projectex-fabric-26.2-1.1.1.jar`.
