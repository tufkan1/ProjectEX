# ProjectEX 26.3 Snapshot 1-4 Patch 2

This prerelease rebuild applies the ProjectEX 1.1.4 menu, inventory, and placed-block
rendering corrections to all four supported Minecraft 26.3 snapshot profiles.

## Included profiles

- Minecraft 26.3 Snapshot 1
- Minecraft 26.3 Snapshot 2
- Minecraft 26.3 Snapshot 3
- Minecraft 26.3 Snapshot 4

Each profile includes the runtime, sources, and Javadoc JARs plus a CycloneDX SBOM and
SHA-256 manifest. All 54 required game tests pass independently on every profile.
