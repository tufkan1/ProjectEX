# ProjectEX 1.1.5

ProjectEX 1.1.5 adds the missing player-control and client-preference layer for Minecraft
26.2 while preserving the server-authoritative gameplay model.

The default bindings are `V` for charge, `G` for mode, and `K` to open an Arcane Tablet
from the inventory. All three appear in Minecraft's normal **Settings > Controls > Key
Binds > ProjectEX** category and can be rebound or cleared. Tablet opening is checked by
the server against the player's current inventory.

Installing optional Mod Menu 20.0.1 adds **Mods > ProjectEX > Configure**. Its vanilla
settings screen controls EMC tooltips, compact EMC display, remembered transmutation
favorites, and automatic search focus. These choices are client-only and remain available
in `config/projectex/client.properties` without Mod Menu.

The release adds automated rendering traversal for every currently registered collector,
relay, storage, condenser, and furnace screen, plus the settings screen. It also adds a
dedicated production boot gate for Mod Menu alongside the existing no-integration and Jade
profiles.

Back up worlds before changing any mod set. This release remains save-compatible with the
stable ProjectEX 1.x line for Minecraft 26.2.
