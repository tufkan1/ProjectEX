# ProjectEX 1.1.4

ProjectEX 1.1.4 restores source-accurate ProjectE and ProjectExpansion menu geometry,
inventory behavior, and placed-block rendering for Minecraft 26.2.

## Highlights

- Removes the synthetic storage pagination and restores each machine's native slot grid.
- Restores source layouts for Energy Condensers, Alchemical Chests, Alchemical Bags,
  Energy Collectors, and Anti-Matter Relays.
- Energy Condenser targets and collector targets now behave as non-consuming templates.
- Energy Condenser MK3 exposes its 91-slot input view and 180-slot output view from the
  appropriate block faces.
- Removes non-source public/private and redstone controls that overlapped original textures.
- Restores the compact ProjectE transmutation panel and original result-ring coordinates;
  transmutation keeps only its source-provided result pagination.
- Expansion relays, power flowers, EMC links, and transmutation interfaces no longer open
  invented container screens.
- Corrects placed-block model inheritance for condensers and alchemical chests.
- Adds automated slot-count, placement, block-entity, model-parent, texture, and blockstate
  integrity coverage. All 54 required game tests pass.

The same fixes are included in the rebuilt Minecraft 26.3 Snapshot 1-4 prerelease artifacts.
