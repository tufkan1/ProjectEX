# ProjectEX 0.1.0-alpha.1

This is the first public ProjectEX alpha for Minecraft Java Edition 26.2 and Fabric.
EMC, transmutation, collectors/relays, condensers, alchemical storage, matter
progression, furnaces, knowledge sharing, configuration migrations, and the public
integration API are available for testing.

## Compatibility

- Minecraft 26.2
- Fabric Loader 0.19.3 or newer
- Fabric API 0.155.2+26.2
- Java 25
- Optional Jade 26.2 integration

EMI does not currently publish a Fabric 26.2 artifact, so EMI integration is not
included in this release. ProjectEX remains fully usable without optional mods.

## Alpha warning

Back up worlds before installing or upgrading. This is a public alpha, not the 1.0
world-stability release. Please report defects using the repository issue templates
and include the generated diagnostics requested by the template.

## Supply-chain verification

The release contains the runtime, source and Javadoc jars, a CycloneDX 1.6 JSON
SBOM, and `SHA256SUMS`. GitHub-hosted provenance and SBOM attestations can be checked
with:

```text
gh attestation verify projectex-fabric-0.1.0-alpha.1.jar --repo tufkan1/ProjectEX
```

See `CHANGELOG.md` for the complete feature list and the player/server guides for
installation, backup and migration instructions.
