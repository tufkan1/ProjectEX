# Support

- Bugs: use the bug report form with full logs and reproduction steps.
- Feature ideas: use the feature request form and describe the gameplay problem.
- Port/parity work: use the port task form with exact upstream references.
- Questions and design discussion: use GitHub Discussions once enabled.
- Security: follow `SECURITY.md`; never publish exploit details in an issue.

Search existing issues first and test the newest ProjectEX release on the exact
Minecraft version it targets. ProjectE and ProjectExpansion maintainers do not support
this project; do not contact them about ProjectEX bugs.
